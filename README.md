# LittleLemon

## Crear proyecto

```powershell
> python3 -m venv venv # crear venv
> source venv/bin/activate # activar venv
> pip install -r requirements.txt # instalar librerias requeridas
> pip install --upgrade pip # actualizar pip

## ---- Projecto de django ----
> django-admin startproject LittleLemon . # crear proyecto de django
> python3 manage.py startapp Restaurant # Crear app Restaurant
> python3 manage.py makemigrations # migraciones iniciales
```
